# YouTube Downloader

برنامج بسيط لتحميل فيديوهات YouTube بجودات مختلفة (1080p / 720p / 480p / 360p / mp3) — يشتغل في المتصفح على جهازك.

---

## طريقة التشغيل (3 خطوات بس)

### 1) حمّل المشروع
اضغط على زرار **Code** الأخضر فوق ← **Download ZIP**، وبعدين فك ضغط الملف.

### 2) دبل-كليك على `setup.bat`
الملف ده بيعمل كل حاجة لوحده (مش محتاج تنزّل أي حاجة تانية):
- ✓ يثبّت Python لو مش متثبّت عندك
- ✓ يثبّت المكتبات المطلوبة
- ✓ ينزّل ffmpeg
- ✓ يبني ملف `YouTube-Downloader.exe`

سيب الشاشة السوداء مفتوحة لحد ما تشوف كلمة **DONE** (حوالي 5-10 دقايق حسب سرعة النت).

### 3) دبل-كليك على `YouTube-Downloader.exe`
خلاص! البرنامج هيفتح في المتصفح تلقائياً.

> 💡 ممكن تنقل ملف `YouTube-Downloader.exe` لأي مكان (Desktop مثلاً) وتشتغل به من هناك مباشرة.

---

## للإيقاف
اقفل النافذة السوداء اللي بتظهر مع البرنامج.

---

## ملاحظات

- لو Windows Defender أو الأنتي-فايرس عمل تنبيه على الملف، ده طبيعي مع البرامج المبنية بـ PyInstaller — اضغط **Run anyway** أو ضيفه للاستثناءات.
- لإعادة البناء: امسح `YouTube-Downloader.exe` وشغّل `setup.bat` تاني.
- مساحة فاضية مطلوبة أثناء البناء: حوالي 2 GB.

---

## للمطورين

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
# ضع ffmpeg.exe و ffprobe.exe في bin/
python app.py
```
