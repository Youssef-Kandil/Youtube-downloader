# YouTube Downloader

برنامج بسيط لتحميل فيديوهات YouTube بجودات مختلفة (1080p / 720p / 480p / 360p / mp3) — يشتغل في المتصفح على جهازك المحلي.

---

## طريقة التشغيل (لو مش مبرمج)

### 1) حمّل المشروع
اضغط على زرار **Code** الأخضر فوق ← **Download ZIP**، وبعدين فك ضغط الملف.

### 2) ثبّت Python (مرة واحدة فقط)
لو مش متثبت عندك:
- روح على https://www.python.org/downloads/
- نزّل آخر إصدار وثبّته
- **مهم جداً**: أثناء التثبيت، فعّل علامة **"Add Python to PATH"**

### 3) شغّل setup.bat
دبل-كليك على ملف `setup.bat`. هيعمل التالي تلقائياً:
- يثبّت مكتبات Python المطلوبة
- ينزّل ffmpeg (لازم لدمج الفيديو والصوت)

(الخطوة دي بتاخد دقيقتين-تلاتة حسب سرعة النت، ومرة واحدة بس.)

### 4) شغّل البرنامج
دبل-كليك على `run.bat` — هيفتح البرنامج في المتصفح تلقائياً على:
**http://127.0.0.1:5000**

---

## كل ما عايز تشغل البرنامج بعد كده
بس دبل-كليك على `run.bat`. ✓

## للإيقاف
اقفل نافذة الـ Command Prompt السوداء.

---

## للمطورين

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
# ضع ffmpeg.exe و ffprobe.exe في bin/
python app.py
```
